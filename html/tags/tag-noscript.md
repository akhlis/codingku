# HTML `<noscript>` Tag

Menentukan tampilan konten saat browser tidak mengaktifkan Javascript.

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
	</div>
</div>

```html
<noscript>Please enable JavaScript.</noscript>
```

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
	</div>
</div>

```html
<noscript>
  <style type="text/css">
    .no-js {
      display: block;
    }
  </style>
</noscript>
```
