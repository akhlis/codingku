# HTML `<canvas>` Tag

Menentukan sebuah elemen yang bisa digunakan untuk menggambar grafis.

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
        <canvas>
        Fallback text for non-supported browsers
        </canvas>
	</div>
</div>

```html
<canvas>
  Fallback text for non-supported browsers
</canvas>
```
