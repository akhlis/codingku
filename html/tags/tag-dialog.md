# HTML `<dialog>` Tag

Defines a dialog box that can be opened and closed with JavaScript.

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
        <dialog>
        This is a dialog box!
        </dialog>
	</div>
</div>

```html
<dialog>
  This is a dialog box!
</dialog>
```
