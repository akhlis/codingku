# HTML `<caption>` Tag

Digunakan untuk membuat judul sebuah `<table>`

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
        <table>
        <caption>The Beatles</caption>
        <tr>
            <td>John Lennon</td>
            <td>Rhythm Guitar</td>
        </tr>
        <tr>
            <td>Paul McCartney</td>
            <td>Bass</td>
        </tr>
        <tr>
            <td>George Harrison</td>
            <td>Lead Guitar</td>
        </tr>
        <tr>
            <td>Ringo Starr</td>
            <td>Drums</td>
        </tr>
        </table>
	</div>
</div>
