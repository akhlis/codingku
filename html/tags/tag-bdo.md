# HTML `<bdo>` Tag

Memungkinkan untuk mengganti teks yang berlawanan arah.

<div class="example">
	<p class="example__label"><strong>Contoh:</strong></p>
	<div class="example__preview">
		The word <bdo dir="rtl">Hello</bdo> is "Hello" spelled backwards.
	</div>
</div>

```html
The word <bdo dir="rtl">Hello</bdo> is "Hello" spelled backwards.
```
